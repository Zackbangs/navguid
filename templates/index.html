NavGuid

A voice-first assistive navigation prototype for blind and visually impaired users. NavGuid supports indoor guidance, outdoor walking navigation, and live hazard awareness.

Voice commands you can try:

Open Navigation Screen
Voice-first control
Users can begin navigation by speaking instead of relying on visual buttons.
Outdoor guidance
Uses GPS and route instructions to support spoken walking navigation.
Hazard awareness
Camera-based detection supports real-time obstacle and hazard alerts.
Voice is the main experience. The buttons are kept only as a fallback when the browser requires one first tap to allow microphone access.
